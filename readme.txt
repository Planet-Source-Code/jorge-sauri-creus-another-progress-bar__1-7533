This code is a customized Progress bar.
You can use their properties and methods to change the image of the bar to the images loaded into the app resource , or you can open an image from a file (bmp, gif, jpeg).
Create a new vb proyect and, then add the progressbar.ocx component to the tools bar, then just use it into your proyect.
To see the complete properties and methods of the activex, open the progressbar code.
